== Description ==

Add to your BuddyPress community the ability to provide reviews and star ratings for the members.  

**[Fast Demo](https://www.wordplus.org/members/admin/reviews/) | [Feature Request](https://www.wordplus.org/forums/forum/bp-user-reviews/)**

Plugin features:

* Show member rating on member profile, member directory, bbpress reply
* Google Rich Snippets Enabled
* Specify who can left reviews (anyone or registered only)
* Specify how many stars is the rating out of
* Specify rating stars colors
* Specify if text review is required alongside with the rating
* Specify if multiple reviews allowed
* Specify single or multiple rating criteria
* Auto spam filtering if Akismet enabled

Languages:

* English
* Norwegian by [@thunoldolsen](https://profiles.wordpress.org/thunoldolsen)
* Danish by Jonas
* Russian

**This is a new plugin, so please use support forums if you have found any bug or have any other problem with plugin**

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/bp-user-reviews` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the BP User Reviews -> Plugin Name screen to configure the plugin
